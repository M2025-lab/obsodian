# 我建议增加一条最重要的学习要求（很多人都忽略）

你未来想做的是 **AI Agent 工程师**，而不是 AI 使用者。因此，**每天除了学一个知识点，还要读源码**。

我建议每天固定增加 **30 分钟源码阅读**，并且按照这个顺序：

|周次|阅读源码|重点关注什么|
|---|---|---|
|第1周|Spring AI Alibaba|ChatClient 是如何发送请求的？AutoConfiguration 如何自动装配？|
|第2周|OpenAI Java SDK|Function Calling、Request/Response 对象如何设计？|
|第3周|Spring AI Alibaba Memory|ChatMemory、Advisor、Memory 的实现方式|
|第4周|一个完整的 Agent 开源项目（如 Spring AI Alibaba Examples）|整体架构、模块划分、扩展点、工程组织|

对于每个类，不要一行一行读，而是固定回答四个问题：

1. **它解决什么问题？**
    
2. **它的输入和输出是什么？**
    
3. **为什么这样设计，而不是另一种方式？**
    
4. **如果让我自己写，我会怎么实现？**
    

这会让你不仅会使用框架，还能理解框架设计思路，为后续学习 LangGraph、MCP、RAG 和企业级 Agent 平台打下非常扎实的基础。
