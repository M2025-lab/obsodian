# 家装材料履约系统（edar-starlord）职责全面分析

---

## 一、项目定位

edar-starlord 是家装行业的**材料履约核心系统**，负责管理从测量、设计、下单、送货到安装验收的全流程材料履约业务。它是连接供应链、施工、服务商的中枢神经。

---

**任务状态流转**:
```
未激活(1) → 激活未完成(2) → 激活已完成(3)
                ↓
           暂停激活(4)
```

###  材料流程配置模块

**核心职责**: 定义材料履约的业务流程模板

- **流程定义**: 通过 `processCode` 关联流程模板
- **节点路由**: 定义节点间的流转关系（测量→设计→复尺→报价变更→送货→安装）
- **激活条件**: 配置节点激活的前置条件（前置节点完成、施工包节点状态等）
- **时间计算**: 配置各节点的考核时间计算规则

**三表关系**:
```
delivery_flow_rule (主表)
    ├── delivery_flow_rule_unit (N:1) - 单据/业务/分公司/套餐组合
    └── delivery_flow_rule_category (N:1) - 品类/供应商/节点流程
```

## 与其他系统的交互

### 上游系统（调用方）

|------|---------|
| 管家APP | 查询任务列表、处理任务 |
